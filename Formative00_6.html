Exercise 1 — Install and verify WSL2 (Required)

Goal

Install WSL2 and Ubuntu.

Steps (PowerShell as Administrator)

wsl --install
wsl --set-default-version 2

If Ubuntu is already installed, you can skip installation and just verify your version.

Verify

wsl --status
wsl -l -v

Exercise 3 — GPU visibility (Required)

Exercise 4 — Python environment for fine-tuning (Required)

Goal

Create a clean Python environment for fine-tuning.

Steps

  1. Install Miniforge (recommended) or Miniconda inside Ubuntu.
  2. Create a new environment named wsu_mis_ft:
conda create -n wsu_mis_ft python=3.11 -y
conda activate wsu_mis_ft
  1. Install core libraries (example set):
python -m pip install --upgrade pip
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
pip install transformers datasets accelerate peft trl bitsandbytes safetensors

If you are CPU-only, use the CPU PyTorch install instructions provided by the instructor.
