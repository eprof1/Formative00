You are expected to have a working knowledge of the following practices and utilities.
If you do not understand how to use an item, perform a Google Search, consult Generative AI, refer to the WSU Technology Knowledge Base,  ask a teaching assistant, or ask the instructor:

  1. Restart laptop on campus before class (Press Ctrl + Shift + Esc, select Performance tab)
  2. Microsoft Windows 10 or 11-installed with latest updates
  3. Set primary email address to Winona State University
    1. Go to the StarID self-service site
    2. Select the 'Sign in to Profile' tab
    3. Login with your StarID and password
    4. Click the 'Office 365 Contact' tab
    5. Click the 'Select New Student Contact Card' button
    6. Select the corrrect version of your Winona State email address
      1. CORRECT-->firstname.lastname @go.winona.edu
      2. DO NOT SELECT--> starID@go.winona.edu
  4. Install and configure the Outlook Desktop Client on your laptop.
  5. Install and configure One Drive for Business.
    Your username is: StarID@go.minnstate.edu
    If you encounter problems, search online for, and install the latest version of "Microsoft Support and Recovery Assistant for Office 365"
    Again, login using your StarID@go.minnstate.edu
    See Tech Support in Somsen 207 if you still have problems.
  6. Log into your LinkedIn Learning account. (More info here)
  7. File Explorer-to create and modify files and folders.
  8. Libraries-to organize your work into virtual folders.
  9. Context sensitive shortcut menus are activated by right-clicking various objects in a window.
  10. Tool bars, menu bars, status bars can be turned on and off using context sensitive menus.
  11. Use of common shortcuts , these will often work even when the menu options will not-
    1. Ctrl+A to select All
    2. Ctrl+C to Copy
    3. Ctrl+V to Paste
    4. Ctrl+Z to Undo
    5. Windows+L to lock screen
    6. Windows Key
  12. Use of the help screen to learn terminology
  13. Appropriate browser/application pairings. Note these can and do change!
    1. Chrome-Google Drive, YouTube, D2L
    2. FireFox-good for privacy
    3. Edge-One Drive for Business, Office365
  14. Useful browser add-ins: Chris Pedrick's Web Developer Toolbar for FireFox
  15. Know basic HTML tags
    1. in FireFox menu select View>Page Source
    2. refer to W3Schools
  16. Validating HTML code-refer to the W3C Markup Service
  17. Using browser to print web pages as .pdf file
    1. Make sure url appears in the Header or Footer-
      see Firefox example
      see Chrome example
  18. Troubleshooting advice-at first try finding a solution yourself
    1. Learn to Google any error message before asking anyone else
    2. Refer to LinkedIn Learning
    3. Learn to use Stack Overflow
    4. Learn to use Tutor.com; see D2L for more info
  19. When seeking advice from teaching assistants or professor via email use proper etiquette
    1. subject line identifying course, section, assignment
    2. use accurate terminology
    3. use complete sentences and proofread
    4. provide screenshots of error messages-use the Snipping Tool
